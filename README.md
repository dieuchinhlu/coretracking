# Getting Started
- Clone the repo using the command `git clone https://github.com/jackyphuong/core-logistictracking.git` and check out the `master` branch
- Run `npm install`
- Run `typings install`
- Run `gulp`
- Run `gulp serve`
- Then open the `http://localhost:3000` url from browser
